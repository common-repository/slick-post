Slick Post show the recent post after scroll and show the recent post.

== Description ==

Slick Post is a simple plugins to show the post after scroll the page. Automatically show the post.

Major features in Slick Post include:

* Automatically Show the page in four position top left, top right, bottom left and bottom right.
* You can select this for custom post type.
* You can select the category of the post.
* Dark theme and light theme is included. 
* Increase the user engaged to your site.
* For pomotional post you can use this.
* Increase the visitor to your blog.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/slick-post` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Plugin Name screen to configure the plugin
1. Check The Setting from the Settings->Slick Post  

1, 2, 3: You're done!
